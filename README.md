# DevBank - Front

Projeto desenvolvido como parte do Trainee Front-end da Dev. Community Mauá, com o objetivo de criar uma aplicação que consome a API do DevBank, um banco virtual fictício.

## 📝 Descrição
O DevBank Front é um site que permite aos usuários interagir com um sistema bancário virtual, realizando operações como:
- 💰 Depósitos
- 💸 Saques  
- 📊 Visualização de histórico de transações

Desenvolvido com **React** e **Vite**, seguindo as especificações da Dev. Community Mauá.

## ✨ Funcionalidades
| Tela | Descrição |
|------|-----------|
| **🏠 Inicial** | Insere URL da API para acesso |
| **📊 Conta** | Mostra saldo e opções (depósito/saque/histórico) |
| **⬆️ Depósito** | Calcula valor total das notas selecionadas |
| **⬇️ Saque** | Similar ao depósito, para retiradas |
| **🕒 Transações** | Exibe histórico completo |
